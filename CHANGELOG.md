# 更新日志

## [1.0.0] - 2024-02-11

### 🎉 初始发布

#### ✨ 新功能
- 基础阅读器功能
  - 支持章节浏览和正文阅读
  - 自动保存阅读进度
  - 侧边章节目录导航
- 书架管理
  - 添加/移除书籍
  - 按阅读时间/添加时间排序
  - 长按移除操作
- 搜索功能
  - 书名/作者搜索
  - 热门榜单展示
  - 搜索结果列表
- 书源管理
  - 兼容 Legado 书源格式
  - 支持多书源切换
  - 书源导入/导出
  - 书源启用/禁用
- 阅读设置
  - 字体大小调节
  - 行高调节
  - 多种背景主题
- 日志系统
  - 应用运行日志
  - 错误追踪
  - 调试信息

#### 🛠️ 技术特性
- Flutter 跨平台支持 (Android, iOS, Windows, macOS, Linux, Web)
- Provider 状态管理
- Material Design 3 UI 设计
- 章节预加载优化
- HTTP 请求重试机制
- 章节内容缓存

#### 📝 文档
- 完整的 README 文档
- 签名配置指南
- 书源开发指南
- 快速开始教程

## 变更说明

本项目的版本号遵循 [语义化版本规范](https://semver.org/lang/zh-CN/)。

- **主版本号**: 重大功能变更或不兼容的 API 修改
- **次版本号**: 向下兼容的功能性新增
- **修订号**: 向下兼容的问题修正
